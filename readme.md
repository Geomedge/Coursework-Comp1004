# Event Calendar
This project uses the Google Calendar API to allow for importing of Events. You will need to write the client id and API key.

## Usage
To start using the program you can choose to import calendar events from a JSON file or use the Google Calendar API with the Authorise login button. Once that happens you will see 3 windows showing 3 of the most recent upcoming events. Additionally the calendar will mark blue squares, which are clickable, which display info of what is happening on that day. 

## Features
- Import / Export JSON calendars
- Urgency system on upcoming events (green means they are more than 2 days away, red means they are in the next 24 hours).
- Interactable monthly calendar to more visually show the events that are coming up.
